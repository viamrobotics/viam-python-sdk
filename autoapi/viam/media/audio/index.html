class viam.media.audio.AudioCodec[source]

Bases: str, enum.Enum

Common audio codec identifiers.

These constants represent commonly supported audio codecs for audioin and audioout components.

Example:

from viam.components.codecs import AudioCodec
from viam.proto.common import AudioInfo

audio_info = AudioInfo(
    codec=AudioCodec.PCM16,
    sample_rate_hz=44100,
    num_channels=2
)
